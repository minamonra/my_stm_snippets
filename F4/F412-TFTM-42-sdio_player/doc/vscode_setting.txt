Сборка и управление проектом
Ctrl + Shift + B — Запустить сборку проекта (вызов make).
F7 — Быстрая сборка (работает во многих раскладках по умолчанию для C/C++ расширений).
Отладка (Cortex-Debug)
F5                —   Главная кнопка: автоматически запускает таску build, проверяет код, компилирует изменения,
                      запускает OpenOCD, прошивает чип через DAP-Link и останавливается на начале функции main().
Shift + F5        —   Полностью остановить отладку и отключиться от платы.
Ctrl + Shift + F5 —   Перезапустить микроконтроллер с самого начала (Reset).
F11               —   Шаг с заходом внутрь функции (Step Into).
F10               —   Шаг без захода внутрь функции (Step Over).
Shift + F11       —   Шаг наружу из текущей функции (Step Out).
F9                —   Поставить или убрать точку останова (Breakpoint) на строке с курсором.Прошивка и стирание
                      без отладки (из вашей панели Tasks)

Если отладка не нужна, а нужно просто запустить утилиты OpenOCD из tasks.json:
Нажмите Ctrl + Shift + P (командная строка VS Code).
Введите Tasks: Run Task (или просто Таска).
Выберите из списка нужную команду: flash, flash-nr или erase-chip
