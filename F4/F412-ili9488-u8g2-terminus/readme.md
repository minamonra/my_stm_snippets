# Шрифт Terminus с кириллицей для дисплея ILI9488 на STM32F412

В этом репозитории представлен пример вывода крупного моноширинного шрифта с поддержкой русского языка. Проект адаптирован под графическую библиотеку U8g2 и цветной TFT-дисплей ILI9488 (разрешение 480x320 пикселей).

![Демонстрация работы интерфейса Terminus на ILI9488](2026-06-05-15-01-43-589.jpg)

## Особенности шрифта

Стандартный набор шрифтов библиотеки U8g2 содержит мало вариантов крупной кириллицы. Для этого проекта были взяты растровые исходники шрифта Terminus, который хорошо подходит для текстовых интерфейсов.

### Основные характеристики Terminus 28b

* **Четкие линии:** Шрифт растровый (Bitmap). Каждая буква отрисована под пиксельную сетку дисплея, поэтому текст рендерится без размытия краев на цветной матрице.
* **Компактность по ширине:** Буквы вытянуты по вертикали и сужены по бокам. Набор символов занимает меньше места в строке, чем стандартный шрифт Inconsolata (`inr24`). Это позволяет уместить больше текста на экране.
* **Поддержка буквы Ё:** В кодовую таблицу включены заглавная `Ё` и строчная `ё` в общих пропорциях шрифта.

## Источники и конвертация

Исходные файлы в формате `.bdf` взяты из открытого репозитория [Tecate/bitmap-fonts](https://github.com).

Сборка Си-массивов выполнялась консольной утилитой `bdfconv.exe` из состава библиотеки U8g2. Чтобы массив не занимал много Flash-памяти микроконтроллера, при генерации были выбраны только два диапазона — базовая латиница (ASCII) и кириллица:

```bash
bdfconv.exe -v -f 1 -m "32-127,1024-1119" ter-u28b.bdf -n u8g2_font_terminus_28b_cyr -o terminus_28b_cyr.c
```

### Как использовать в проекте

1. Скопируйте файл `terminus_28b_cyr.c` в папку вашего проекта.
2. В коде приложения объявите внешний массив шрифта: `extern const uint8_t u8g2_font_terminus_28b_cyr[];`.
3. Очистите каталог и соберите проект через терминал:

```bash
make clean
make flash
```
